# hog

Find out who hogs (memory|CPU) on your system.

## Installation

If you have `npm` installed, do:
```
npm install -g hog
```

## Usage
```
hog [cpu|mem]
```

Searches for clutters. Per default, it searches for memory hogs 
(I'm looking at you, Firefox).
